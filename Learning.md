# Learnings from Task 1

## What I Learned

### Table

A table stores records in ServiceNow.

Example:
incident

### Record

A single row inside a table.

Example:
One Incident record.

### Field

A column that stores information.

Example:
Caller
Category
Priority

### Dictionary Entry

Defines field properties and metadata.

Stored in:
sys_dictionary

### Choice Value

Stores dropdown options.

Stored in:
sys_choice

### Form Layout

Controls which fields appear on a form and their order.

### Custom Fields

Custom fields created by users generally begin with:

u_

Example:

u_sfone_model

### Verification

Verified that:

* Custom field appeared on the Incident form.
* New category choice appeared in dropdown.
* Incident record was successfully created.
